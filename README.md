# 📖 Overview

The Gym Management System is a Java desktop application designed to streamline the management of gym members through a clean and interactive graphical user interface.

The application enables gym staff to register members, manage membership information, calculate membership fees, and maintain member records without relying on manual paperwork.

The project was developed using Java Swing while following Object-Oriented Programming principles to ensure modularity, maintainability, and code reusability.

---

# ✨ Features

- 🏋️ Register Regular Members
- 👑 Register Premium Members
- ✏️ Update Member Information
- 💰 Membership Fee Calculation
- 📅 Membership Plan Management
- 📋 Display All Member Details
- ❌ Remove Member Records
- ✔️ Input Validation
- 🖥️ Interactive Java Swing GUI
